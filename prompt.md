# Website Prompt for Global CR Transport

Create a professional, responsive logistics web application for **Global CR Transport**, a company with **20 years of experience** in international freight forwarding (import/export) based in Costa Rica.

### Key Requirements:
1.  **Bilingual Support**: The entire site must be available in both **English and Spanish** with a visible language toggle in the navigation bar.
2.  **Public Tracking System**: 
    - A prominent search bar on the homepage for customers to enter a Tracking Number.
    - A dedicated tracking results page that displays shipment details (Origin, Destination, Status, ETA) and a visual progress timeline.
3.  **Visual Style**:
    - **Theme**: Deep Navy Blue and Professional White (Corporate Logistics feel).
    - **Typography**: Clean, modern sans-serif fonts (e.g., 'Outfit' for headings, 'Inter' for body).
    - **Imagery**: High-quality visuals of cargo ships, planes, and land transport.
4.  **Floating Support**: A persistent, floating **WhatsApp button** in the bottom right corner linked to `+506 2222-2222` for operations support.
5.  **Service Highlights**:
    - Section showcasing: Ocean Freight, Air Freight, and Land Transport.
    - Mention "20 Years in the Market" as a trust indicator.
6.  **Responsive Design**: The layout must be fully optimized for mobile, tablet, and desktop devices.
7.  **Tech Stack**:
    - **Frontend**: React with Tailwind CSS and shadcn/ui components.
    - **Icons**: Lucide-react for consistent iconography.
    - **Animations**: Subtle fade-in and hover transitions using Framer Motion.

### Content Focus:
- **Headline**: "Connecting The World" / "Conectando El Mundo".
- **Mission**: Provide premium logistics solutions for Costa Rica and the world with precision, care, and speed.
- **Trust Indicators**: Secure Shipping, Global Reach, 24/7 Support.
